# 변수와 자료형 study at 6/27

## 변수 (Variable)
데이터를 저장하는 메모리 공간에 붙여준 이름
~~~java
int age = 20;
String country = "Korea";
~~~

## 변수 이름 규칙

1. 문자와 숫자, _, $ 사용 가능
~~~java
int apple = 2000;
int apple3 = 6000;
int _apple = 2000;
~~~
2. 숫자로 시작은 불가능!
~~~java
int 3apple = 6000; // 에러 발생
~~~
3. 대문자와 소문자는 구분
~~~java
// 서로 다른 변수로 취급됨
class exam3 {
    public static void main(String[] args) {
        int apple = 2000;
        int Apple = 3000;
        int APPLE = 2000;

        System.out.println(apple);
        System.out.println(Apple);
        System.out.println(APPLE);
    }
    
}
~~~
4. 공백 사용 X
~~~java
int one apple = 2000;
~~~
5. 미리 예약된 이름 사용 X
~~~java
// 자바에서 사용되는 예약어는 변수로 취급 x
int true = 1; 
int if = 2;
int continue = 10;
~~~

## 기본형과 참조형

기본형 변수: 실제 값(data)을 저장한다. Ex) int a = 0;

참조형 변수: 어떤 값이 저장되어 있는 주소를 값으로 갖는다. 기본형 변수를 제외한 모든 타입의 변수이다. Ex) String s = "Hello"; // 

## 표기법

* 카멜 표기법 (camelCase): 가장 앞의 문자는 소문자로, 나머지 단어의 첫 문자는 대문자로 표기
~~~java
// 변수명이나 메서드 이름을 표기할 때 사용
int oneApple = 1000; 
String firstName = "LEE";

public void setValue(int value) {
    this.value = value;
}
~~~
* 파스칼 표기법: 각 단어의 첫 문자를 대문자로 표기
~~~java
class Main{ // 대부분 클래스 이름을 표기할 때 사용
    
}
~~~

## 자료형

* 정수: 양의 정수와 음의 정수를 나타내는 자료형 (short, int, long)
~~~java
int intNum = 10;
long longNum = 21474836L;
~~~
* 실수: 소수를 나타내는 자료형(float, double)
~~~java
float floatNum = 0.1f;
double doubleNum = 3.4028236E38;
~~~
* 부울(Boolean): 참과 거짓을 나타내는 자료형(boolean)
~~~java
boolean isPass = true; // 참
boolean isOk = false; // 거짓
~~~
* 문자(Character): 한 개의 문자를 나타내는 자료형(char)
~~~java
char keyFirst = 'a';
char keyLast = 'z';
~~~
* 문자열(String): 2개 이상의 문자들로 이루어진 집합
~~~java
String s1 = "Hello World!";
String s2 = "01234";
~~~
* StringBuffer: 문자열을 자주 추가하거나 변경할 때 사용하는 자료형
~~~java
StringBuffer sb1 = new StringBuffer("Hello World!");
~~~
* 배열(Array): 자료형이 같은 많은 수의 데이터를 저장할 수 있는 자료형
~~~java
int[] myArray1 = {1, 2, 3, 4, 5};
char[] myArray2 = {'a', 'b', 'c', 'd', 'e'};
~~~
* 리스트(List): 배열과 같이 여러 데이터를 담을 수 있는 자료형
* 추가로 여러가지 메소드를 제공 (add, get, size, remove, clear, sort, contains)
~~~java
ArrayList l1 = new ArrayList();
l1.add(1);
~~~
* 셋(Set): 중복 되지 않고 여러 데이터를 저장하는 자료형
~~~java
HachSet set = new HashSet();
set.add(1);
~~~
* 맵(Map): key, value 형태로 데이터를 저장하는 자료형
~~~java
HashMap<String, String> map = new HashMap<String, String>();
map.put("product", "kiwi");
~~~
* 제네릭스(Generics): 자료형을 명시적으로 지정
* 제한적일 수 있지만 안정성을 높혀주고 형변환을 줄여줌.
~~~java
ArrayList<String> l1 = new ArrayList<String>(); // <>안에 자료형을 입력
HashMap<String, Integer> map1 = new HashMap<String, Integer>();
~~~
