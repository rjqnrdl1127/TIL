# 여러가지 연산자

## 항과 연산자
1. 단항 연산자: 항이 한 개
~~~java
num++
~~~
2. 이항 연산자: 항이 두 개 예) 사칙연산
~~~java
1 + 1
~~~
3. 삼항 연산자: 항이 세 개 예) ? :
~~~java
(3 > 1) ? 1 : 0;
~~~
## 대입 연산자 (=)
* 우측의 데이터를 좌측 변수에 저장
~~~java
int num = 1;
~~~
## 부호 연산자 (+, -)
부호를 나타내는 연산자
## 산술 연산자 (+, -, *, /, %)
* 덧셈(+), 뺄셈(-), 곱셈(*), 나눗셈(/), 나머지(%)</br>
ex) 10 % 3
## 증가/감소 연산자 (++, --)
* 정수 값을 1 증가 또는 감소시키는 연산자 </br>
ex) i++, --i
## 관계 연산자 (>, <, ==, >=, <=, !=)
* 두 항의 값을 비교하는 연산자
## 논리 연산자 (&&, ||, !)
* 논리식에 대해 참 거짓 판단, 결괏 값은 true, false로 나타남.
## 복합 대입 연산자 (+=, -=, *=, /=, %=)
* 대입 연산와 다른 연산자를 조합한 연산자</br>
* 코드를 간결하게 작성할 때 사용
## 비트 연산자 (&, |, ^, ~)
* 비트 단위로 연산
* AND 연산(&): 두 개의 비트가 1인 경우에만 결과가 1
* OR 연산(|): 둘 중 한 개의 비트가 1인 경우 결과가 1
* XOR 연산(^): 두 개의 비트가 다르면 0, 같으먄 1
* 반전 연산(~): 비트가 0이면 1, 1이면 0으로 반
## 비트 이동 연산자 (>>, <<, >>>)
* 비트를 오른쪽 또는 왼쪽으로 이동시키는 연산자
